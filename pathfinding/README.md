Project Description
This project is a Pathfinding Algorithm implementation using Python that finds the shortest path between a start point and a goal point in a grid environment.
The system simulates how navigation systems and robots find optimal paths by avoiding obstacles and exploring possible routes efficiently.
The main objective of this project is to understand and implement graph traversal and shortest path algorithms using core Python concepts.
Features:
✅ Grid-based environment for pathfinding
✅ Support for obstacles and blocked paths
✅ Shortest path calculation from source to destination
✅ Implementation of Breadth First Search (BFS) algorithm
✅ Step-by-step path reconstruction
✅ Command-line based execution
Technologies Used:
Python
Data Structures (Queue, Lists, Matrices)
Graph Algorithms
Breadth First Search (BFS)
How It Works:
The grid is represented as a matrix where free cells and obstacles are defined.
The algorithm explores neighbouring cells using BFS to ensure the shortest path is found.
Once the goal is reached, the path is reconstructed and displayed.
